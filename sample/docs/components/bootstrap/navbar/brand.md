<abt-navbar expand-size="lg" navbar-color-type="light" background-color-type="light">

You can also use `heading` instead of `heading="true"`.
```html
<abt-navbar expand-size="lg" navbar-color-type="light" background-color-type="light">
  <abt-navbar-brand href="#">NavBar</abt-navbar-brand>
</abt-navbar>
<abt-navbar expand-size="lg" navbar-color-type="light" background-color-type="light">
  <abt-navbar-brand class="mb-0 h1" heading>NavBar</abt-navbar-brand>
</abt-navbar>
```
