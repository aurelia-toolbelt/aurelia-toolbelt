```html
<abt-carousel navigator="true">
  <abt-carousel-image src="/images/pic1.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image src="/images/pic2.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image active="true" src="/images/pic3.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image src="/images/pic4.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image src="/images/pic5.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
</abt-carousel>
```
You can also use `navigator` instead of `navigator="true"` and `active` instead of `active="true"`.
```html
<abt-carousel navigator>
  <abt-carousel-image src="/images/pic1.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image src="/images/pic2.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image active src="/images/pic3.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image src="/images/pic4.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
  <abt-carousel-image src="/images/pic5.jpg" alt="The image cannot be displayed">
  </abt-carousel-image>
</abt-carousel>
```
